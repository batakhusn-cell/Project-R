# Project R 🏁

Project R is a modern streetwear clothing brand inspired by automotive culture, speed, and individuality. Built for car enthusiasts, drivers, and creators who live life in the fast lane.

The “R” stands for **Rogue** — representing those who create their own path and never follow the crowd.

---

## 🚗 About the Brand

Project R blends fashion with car culture. From street racing aesthetics to clean minimal designs, every piece reflects the energy of:
- Night drives 🌃  
- Car meets 🏎️  
- Racing culture 🏁  
- Streetwear lifestyle 🖤  

---

## 🛒 Website Features

- Modern responsive design
- Product catalog (hoodies, tees, accessories)
- Shopping cart system
- Dark themed UI inspired by car culture
- Fast and mobile-friendly experience
